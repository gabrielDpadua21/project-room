## Requirements

Before you start, make sure you have the following installed:

- Docker
- Docker Compose
- Zsh as your terminal
- Python 3.9+

To run the project, simply execute the following command in your terminal:

```bash
bash install
```

See the commands to run

```
room -h
```

### Uninstall

```bash
bash uninstall.sh
```
